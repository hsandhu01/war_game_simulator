# Global Threat Simulator

A browser-based, highly interactive React and D3 simulation engine that strictly models geographical tension, economic state, and threat levels through mathematical calculations.

This project was built securely to represent a fictional geopolitical engine modeled entirely on theoretical math arrays and React state propagation.

## Features
- **Topological D3 Interface**: Completely custom-built mapping logic resolving the entire global geometry (190+ independent pathing entities) mapped strictly onto an interactive SVG viewport.
- **Dynamic Threat Economies**: Background real-time Game Loops (2.5-second cycles) generating AP (Action Points) tied implicitly to relative entity power metrics.
- **Complex AI Logic**: Non-playable objects physically track anger tokens, venting hostility dynamically through an aggression decay algorithm.
- **CovOps & Full Scale Mechanics**: Randomly calculated Espionage, dynamic Defensive Pacts, and Annihilation engines calculate total math success limits.

## Installation

To run this simulation locally:

1. \`npm install\`
2. \`npm run dev\`
3. Open \`http://localhost:5173/\`

Built heavily using React 19, D3-Geo, and custom CSS effects representing CRT Phosphor aesthetics.
